jsvm playground C → WASM

A JavaScript engine written from scratch in C — bytecode compiler, mark-sweep GC, closures, async/await, and ES modules — compiled to WebAssembly. The editor and the REPL on the right share one engine instance: load a sample, run it, then poke at its functions and variables live in the REPL. Everything executes in the C VM, not your browser's JS engine.
